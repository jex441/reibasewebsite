We transform legacy Pull Requests into Web3.

We at Reibase created the protocol Turbosrc. It allows token holders to engage with pull requests. Maintainers create tokens for their project via Turbosrc. They can then distribute or sell those tokens to their communities. Token holders vote on pull requests - accept or deny. Each project (likely maintainers) creates their own set of tokens.
